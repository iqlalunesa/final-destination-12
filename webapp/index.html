Simple Gallery App

This project is a static photo gallery website built with HTML and CSS, combined with a storage usage monitoring system. It displays local images and alerts when disk usage exceeds 80%, with a Grafana dashboard for visualization. The site runs on Docker using Nginx, and updates automatically via a Jenkins CI/CD pipeline. Monitoring is handled by Prometheus Node Exporter.
